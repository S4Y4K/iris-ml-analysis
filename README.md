# Iris Dataset Analysis

This project explores the famous Iris dataset using Python.

## Dataset
The Iris dataset contains measurements of iris flowers including:
- Sepal length
- Sepal width
- Petal length
- Petal width
- Species

## Tools Used
Python
Pandas
Matplotlib
Seaborn

## Analysis Performed
- Data exploration
- Pair plots
- Joint plots
- Correlation heatmap

## Goal
Practice data visualization and understand feature relationships in the dataset.
